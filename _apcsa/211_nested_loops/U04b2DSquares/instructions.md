Try to complete this code in the most efficient way, by using one static method.
  
Write a program that reads user input and displays, using asterisks, a filled and hollow square, placed next to each other. For example, if the side length is 5, the program should display:

```
Enter side length: 5
***** *****
***** *   *
***** *   *
***** *   *
***** *****
```

NOTICE: Your squares MUST be placed next to each other horizontally. Like this...
```
Enter side length: 3
*** ***
*** * *
*** ***
```

NOT THIS!!! Your squares must NOT look like this...
```
***
***  
***

***
* *  // Do NOT do this
***
```

Part 2, ask the user what letter they want to use. 
```
--- Part 2 ---
Enter side length: 4
Which letter to use: x
xxxx xxxx
xxxx x  x
xxxx x  x
xxxx xxxx
```

Part 3, ask the user for the width and height.
```
--- Part 3 ---
Enter width: 5
Enter height: 6
Which letter to use: e
eeeee eeeee
eeeee e   e
eeeee e   e
eeeee e   e
eeeee e   e
eeeee eeeee
```
